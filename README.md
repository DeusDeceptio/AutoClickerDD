Автокликер имеет следующие особенности:
1. 3 режима работы (Клик, Зажатие, Перетаскивание),
2. Можно выбрать какой клавишей будет совершаться действие (ЛКМ, СКМ, ПКМ),
3. Клик по выбранной точке,
4. Настраиваемая горячая клавиша для переключения работы (по умолчанию F6),
5. Выбор задержки между кликами (В милисекундах),
6. Есть режим без задержки, в задержке поставить 0 (может сильно грузить процессор),
7. Закрепление поверх других окон при необходимости.
