# AutoClickerDD

**AutoClickerDD** — это программа для автоматизации действий мыши на Windows. Позволяет автоматически кликать, зажимать кнопки и перетаскивать объекты с настраиваемыми задержками и координатами.

## Возможности
**3 режима работы**:
  - **Кликер** — обычные клики с заданной задержкой
  - **Зажатие** — нажатие и удержание кнопки на заданное время
  - **Перетаскивание** — плавное перемещение мыши с зажатой кнопкой

**Выбор кнопки мыши** — левая, правая или средняя (в режимах "Клик" и "Зажатие" можно выбрать одну кнопку, в остальных — комбинации)

**Координаты**:
- Клик в текущей позиции курсора
- Клик по заданным координатам (с захватом кликом мыши)

**Управление**:
  - Горячая клавиша для включения/выключения (по умолчанию **F6**, настраивается)
  - Закрепление окна поверх других (включено по умолчанию)

**Задержки**:
- Интервал между действиями (мс)
- Время удержания кнопки в режиме "Зажатие" (мс, 0 = держать постоянно)
- Длительность перетаскивания в режиме "Перетаскивание" (мс)

**Дополнительно**:
- Дабл-клик в режиме "Кликер"

## Требования

- Windows 7/10/11
- [.NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework) или выше (обычно уже установлен в Windows)

## Установка
1. Скачайте последнюю версию из раздела [Releases](https://github.com/DeusDeceptio/AutoClickerDD/releases)
2. Запустите `AutoClickerDD.exe` (не требует установки)

## Возможные ошибки
Проект создавался на начальных этапах обучения (2023-2024), поэтому возможны необработанные исключения или неочевидные баги.
